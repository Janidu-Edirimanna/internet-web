INTERNET

The Internet is a global system that connects computers and devices worldwide, enabling communication, information sharing, and access to digital services.

It links billions of devices—like computers, phones, and servers—together using physical cables, optical fibers, and wireless signals, allowing them to communicate and share information instantly.

How the Internet works

The Internet relies on a set of rules called TCP/IP (Transmission Control Protocol/Internet Protocol) to ensure data gets from point A to point B. This infrastructure is largely driven by a few core concepts:



Client-Server Model

The Client-Server Model is a network architecture in which clients send requests for resources or services, and servers process these requests, returning the required responses.

  • Client: A device or program that requests data or services
  • Server: A system that stores resources, manages data, and responds to client requests.
  • Request–response mechanism: Communication follows a structured cycle — client requests, server responds.
  • Centralized management: Data and services are controlled from servers, improving security and consistency.

HTTP (Hypertext Transfer Protocol)

HTTP stands for Hypertext Transfer Protocol, and it’s the system that allows communication between web browsers and web servers.HTTP is a set of rules that lets your browser and web server communicate, ensuring websites load correctly.When you visit a website, your browser uses HTTP to send a request to the server hosting that site, and the server sends back the data needed to display the page.

DNS (Domain Name System)

Translation of human-readable domain names into IP addresses enables computers to communicate over the internet, a function performed by the Domain Name System (DNS), which operates in a hierarchical and distributed manner while improving performance through caching.

  • Provides name-to-address resolution using a hierarchical system (Root, TLD, Authoritative servers) to locate domain information.
  • Improves performance by using caching mechanisms, reducing lookup time for repeated requests

INTERNET IN DAILY LIFE

The Internet has seamlessly integrated into almost every aspect of modern routine:

IN COMMUNICATION

Instant global connectivity via email, video calls, and social media platforms.

IN ECOMMERCE

Online shopping, digital banking, and managing finances instantly without visiting a physical branch.

IN WORK & EDUCATION

Remote working tools, cloud collaboration (like Google Workspace or Microsoft 365), and online learning resources.

WEB BROWSER

Application software that allows users to access, search, and view information on the World Wide Web by communicating with web servers is known as a web browser. It enables smooth navigation of websites by retrieving and displaying online content in an interactive format.

  • Displays web pages with text, images, videos, and links in a user-friendly manner.
  • Sends and receives data using HTTP/HTTPS protocols to load websites securely

Eg : Mozilla Firefox, Microsoft Internet Explorer, Google Chrome, Opera browser, Apple Safari





THE DOMAIN NAME SYSTEM (DNS)

The Domain Name System (DNS) is often called the "phonebook of the Internet."

Computers identify each other on the network using numerical addresses called IP Addresses (e.g., 142.250.190.46). However, these long strings of numbers are nearly impossible for humans to memorize. DNS acts as a directory that maps easy-to-remember domain names (like example.com) to those specific machine-readable IP addresses.

Example: When you enter a website name like google.com in your browser, the system does not understand the name directly. DNS translates it into an IP address of the server hosting the website. This allows your browser to connect to the correct server and load the webpage.
